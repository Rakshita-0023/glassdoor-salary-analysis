# Glassdoor Salary Analysis

##  Project Overview
This project analyzes salary data from Glassdoor to understand compensation trends across different job roles, locations, companies, and experience levels. The objective is to extract meaningful insights from real-world salary data using data analysis and visualization techniques.

The project focuses on data cleaning, exploratory data analysis (EDA), and insight generation to support data-driven decision-making.

---

##  Problem Statement
Salary information available on job portals like Glassdoor is often unstructured and noisy. This project aims to clean and analyze such data to identify patterns in salaries based on factors such as job title, company, location, and experience.

---

##  Dataset
- Source: Glassdoor salary dataset
- Data Type: Tabular
- Features include:
  - Job Title
  - Company Name
  - Location
  - Salary Estimates
  - Experience Level
  - Ratings (where available)

---

##  Tools & Technologies
- **Programming Language:** Python  
- **Libraries:**  
  - Pandas, NumPy (data cleaning & manipulation)  
  - Matplotlib, Seaborn (data visualization)  
- **Environment:** Jupyter Notebook / Google Colab  
- **Version Control:** Git & GitHub  

---

##  Project Workflow

### 1. Data Cleaning
- Removed missing and inconsistent values
- Processed salary strings into numerical formats
- Handled duplicates and invalid entries
- Standardized categorical fields

### 2. Exploratory Data Analysis (EDA)
- Analyzed salary distributions
- Compared salaries across job roles and locations
- Examined the impact of experience on compensation
- Visualized trends using bar plots and distributions

### 3. Insights Generation
- Identified high-paying roles and locations
- Observed salary variation across companies
- Highlighted trends related to experience levels

---

##  Key Insights
- Salary varies significantly based on role and location
- Experience level has a strong impact on compensation
- Certain job titles consistently offer higher salary ranges
